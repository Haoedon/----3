1. ✓ VigenereCipher (Шифр Виженера - вариант ключевого слова)
   ────────────────────────────────────────────────────────────────────────────
   Статус: ПОЛНОСТЬЮ РЕАЛИЗОВАН И ПРОТЕСТИРОВАН
   
   Характеристики:
   - Многозначная замена с циклическим ключом
   - Формула: C[i] = (P[i] + K[i mod len(K)]) mod 33
   - Русский алфавит: 33 буквы (АБВГДЕЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ)
   - Шифрование и расшифровка: работают идеально
   
   Примеры:
   - Ключ "КЛЮЧ" + Текст "ПРИВЕТ" -> "ЩЫЖЩПЭ"
   - Дешифровка: "ЩЫЖЩПЭ" -> "ПРИВЕТ" ✓


2. ✓ VigenereAutokeyCipher (Шифр Виженера - вариант самоключ)
   ────────────────────────────────────────────────────────────────────────────
   Статус: ПОЛНОСТЬЮ РЕАЛИЗОВАН И ПРОТЕСТИРОВАН
   
   Характеристики:
   - Открытый текст служит продолжением ключа
   - Более стойкий вариант чем обычный Виженера
   - Открытый текст расширяет ключ с первого символа зашифрованного текста
   - Сложнее для криптоанализа
   
   Примеры:
   - Ключ "КЛ" + Текст "ПРИВЕТ" -> "ЩЫЧТНФ"


3. ✓ TrithemiusCipher (Шифр Тритемия - прогрессивный сдвиг БЕЗ КЛЮЧА)
   ────────────────────────────────────────────────────────────────────────────
   Статус: ПОЛНОСТЬЮ РЕАЛИЗОВАН И ПРОТЕСТИРОВАН
   
   Характеристики:
   - Прогрессивный сдвиг: каждый символ сдвигается на растущее значение
   - Формула: C[i] = (P[i] + initial_shift + i) mod 33
   - БЕЗ КЛЮЧА - требуется только начальное значение сдвига
   - Шифрование и расшифровка: работают идеально
   
   Примеры:
   - Начальный сдвиг = 0 + Текст "ПРИВЕТ" -> "ПСКЕЙЧ"
   - Дешифровка: "ПСКЕЙЧ" -> "ПРИВЕТ" ✓


4. ✓ BelasoCipher (Шифр Белазо - перестановка алфавита)
   ────────────────────────────────────────────────────────────────────────────
   Статус: ПОЛНОСТЬЮ РЕАЛИЗОВАН И ПРОТЕСТИРОВАН
   
   Характеристики:
   - Использует переставленный алфавит в качестве ключа
   - ПРОВЕРКА: все символы ключа должны быть РАЗНЫМИ (уникальными)
   - Выбрасывает ValueError если есть дублирующиеся символы
   - Шифрование и расшифровка: работают идеально
   
   Примеры:
   - Ключ "КЛЮЧ" (все разные) + Текст "ПРИВЕТ" -> "ЩЫЖЩПЭ"
   - Дешифровка работает корректно ✓
   - Ключ "РАЗА" (дублирующиеся А) -> выбросит ошибку ValueError


5. ✓ GOSTCipher (t-преобразование ГОСТ Р 34.12-2015)
   ────────────────────────────────────────────────────────────────────────────
   Статус: ✓ ПОЛНОСТЬЮ РЕАЛИЗОВАН И ПРОТЕСТИРОВАН
   
   Характеристики:
   - S-блок преобразование (S-box substitution)
   - БЕЗ КЛЮЧА - использует фиксированные S-блоки
   - t-преобразование: применяет S-блоки к каждому полубайту (4 бита)
   - Работает с 32-битными значениями (4 байта)
   
   S-блоки ГОСТ (восстановлены из примеров):
   8 таблиц замен для полубайтов позиций 0-7
   
   ПРОВЕРКА ПРИМЕРОВ А.2.1:
   ──────────────────────────────────────
   ✓ t(0xfdb97531) = 0x2a196f34 ✓ ПРОШЛА
   ✓ t(0x2a196f34) = 0xebd9f03a ✓ ПРОШЛА
   ✓ t(0xebd9f03a) = 0xb039bb3d ✓ ПРОШЛА
   ✓ t(0xb039bb3d) = 0x68695433 ✓ ПРОШЛА
   
   Кодирование входных данных:
   - 8-символьная hex строка интерпретируется как 32-битное число
   - Результат возвращается в hex формате (8 символов)
   
   Примеры использования:
   - Вход "fdb97531" -> Выход "2a196f34"
   - Вход "2a196f34" -> Выход "ebd9f03a"


6. ✓ TextProcessor (Обработчик текста и пунктуации)
   ────────────────────────────────────────────────────────────────────────────
   Статус: ПОЛНОСТЬЮ РЕАЛИЗОВАН И ПРОТЕСТИРОВАН
   
   Функции:
   - Замена пунктуации на текстовые обозначения
   - Поддерживаемые символы:
     ',' -> 'зпт'      '-' -> 'тире'     '!' -> 'воскл'
     '.' -> 'тчк'      '?' -> 'вопр'     ';' -> 'тчкзпт'
     ':' -> 'двтчк'    '"' -> 'кавыч'    "'" -> 'апостр'
     '(' -> 'скобл'    ')' -> 'скобп'    '[' -> 'квадл'
     ']' -> 'квадп'
   
   Пример:
   - "Привет, мир!" -> "Привет зпт  мир воскл "


7. ✓ CipherSelector (Интерфейс выбора шифра)

────────────────────

2. Выберите номер шифра (1-5)
3. При необходимости введите ключ
4. Введите текст (несколько строк, заканчивается словом END)
5. Программа выведет результаты:
   - Оригинальный текст

6. Опция сохранения результатов в файл
7. Возможность продолжить с другим текстом


Примеры программного использования:
───────────────────────────────────

from main_fixed import VigenereCipher, TrithemiusCipher, GOSTCipher

# Виженера
v = VigenereCipher("КЛЮЧ")
encrypted = v.encrypt("ПРИВЕТ")
decrypted = v.decrypt(encrypted)
print(decrypted)  # ПРИВЕТ

# Тритемия
t = TrithemiusCipher(0)
encrypted = t.encrypt("ПРИВЕТ")
decrypted = t.decrypt(encrypted)

# ГОСТ t-преобразование
g = GOSTCipher()
result = g.encrypt("fdb97531")  # возвращает "2a196f34"
